How to Use PBImap

Welcome to the usage instructions for PBImap. Follow these steps to get started with the custom map visual in Power BI.

Step 1: Install the PBImap Visual

Download the PBImap visual from the Downloads page. Once downloaded, open Power BI, go to the "Visualizations" pane, and click on "Import a visual from a file." Select the PBImap visual file to add it to your Power BI report.

Step 2: Configure the Data

  1. Drag the PBImap visual onto your report canvas.
  2. In the "Fields" pane, assign longitude and latitude fields to map data points.
  3. Customize additional data fields as needed for enhanced functionality.

Step 3: Customize the Map

PBImap offers various customization options. Use the "Format" pane to adjust settings such as color, layer opacity, map style, and more to match your report’s design.

Step 4: Interact with the Map

Users can drag and drop map points, zoom, and pan. PBImap is designed to integrate seamlessly with Power BI’s filters and slicers.
